# AdSense Readiness Checklist

기름값 헌터의 애드센스 심사 통과 가능성을 높이기 위한 실제 작업 체크리스트.

## 목표

- 툴 중심 사이트에서 신뢰 가능한 정보 서비스 구조로 확장
- 정책 위반 리스크를 줄이고, 심사자가 검토 가능한 신뢰 요소를 명확히 노출
- 광고를 붙이기 전 콘텐츠와 운영 페이지를 먼저 준비

## 1단계. 신뢰 페이지 구축

- [x] `/about` 추가
- [x] `/contact` 추가
- [x] `/privacy` 추가
- [x] `/terms` 추가
- [x] 모든 페이지에서 정책/문의 링크 접근 가능하도록 푸터 연결

## 2단계. 콘텐츠 자산 확장

- [x] `가이드/콘텐츠 허브` 라우트 설계
- [x] 승인용 콘텐츠 주제 10개 확정
- [x] 1차 콘텐츠 5개 작성
- [x] 2차 콘텐츠 5개 작성
- [x] 지역 랜딩과 콘텐츠 간 내부 링크 연결

권장 주제 예시:

- 오늘 서울 휘발유 최저가 TOP10
- 강남구 경유 가격 비교
- 알뜰주유소와 브랜드 주유소 차이
- 장거리 운전 전 주유 체크리스트
- 전국 평균 유가 읽는 법

## 3단계. 지역 페이지 차별화

- [ ] 지역별 평균 대비 설명 추가
- [ ] 지역별 운전자 맥락 추가
- [ ] 생활권/자주 찾는 구역 차별화
- [x] 관련 가이드 링크 연결

## 4단계. 광고 배치 원칙 수립

- [x] 광고 우선 적용 페이지 선정: `/`, 콘텐츠 페이지
- [x] 광고 제외 페이지 선정: `/gas-finder` 결과 영역, `/admin`
- [x] 버튼/지도 링크/검색 필터 근처 광고 금지
- [x] 첫 배치 위치 2~3개만 실험

## 5단계. 운영 준비

- [x] `ads.txt` 준비 및 배포
- [ ] Search Console 색인 상태 확인
- [ ] 네이버 서치어드바이저 색인 상태 확인
- [ ] 모바일 레이아웃 점검
- [ ] 404/빈 페이지/미완성 페이지 점검

비고:

- `ads.txt`는 `/ads.txt` 경로로 준비되어 있으며, 실제 AdSense 퍼블리셔 ID가 생기면 `GOOGLE_ADSENSE_PUBLISHER_ID`, `GOOGLE_ADSENSE_CLIENT_ID`, `NEXT_PUBLIC_GOOGLE_ADSENSE_CLIENT_ID` 중 하나를 넣으면 활성화됩니다.
- 광고 스크립트와 슬롯은 준비되어 있으며, 실제 노출을 원하면 `GOOGLE_ADSENSE_CLIENT_ID` 또는 `NEXT_PUBLIC_GOOGLE_ADSENSE_CLIENT_ID`와 함께 아래 슬롯 ID 중 필요한 값을 넣으면 됩니다.
- 홈: `GOOGLE_ADSENSE_SLOT_HOME_INLINE`
- 가이드 허브: `GOOGLE_ADSENSE_SLOT_GUIDES_INLINE`
- 가이드 상세: `GOOGLE_ADSENSE_SLOT_GUIDE_DETAIL_INLINE`
- 현재 가이드는 `/guides` 허브와 각 지역 랜딩에서 내부 링크로 연결됩니다.

## 최종 신청 전 체크

- [x] 신뢰 페이지 4개 공개
- [x] 콘텐츠 페이지 10개 이상 공개
- [x] 지역 페이지 내부 링크 보강 완료
- [x] 사이트맵/robots 반영 확인
- [x] 광고 위험 위치 없음
- [x] 운영자 연락 수단 공개
- [x] `ads.txt` 준비 완료
